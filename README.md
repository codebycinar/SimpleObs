# SimpleObs
SimpleObs reposu .Net Core ile hazırlanmış API' lara istek yapan Asp.Net Core arayüzlerden oluşan basit bir öğrenci-not senaryosunun demosudur.

# Kullanılan Teknolojiler

> .Net Core 3.1
> Identity
> EntityFrameworkCore
> SQLite
> Automapper
> Swagger
